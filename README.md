# TranslateTool
### 说明 Qt5实现的翻译小工具
##### 软件开启后，会监听Ctrl+c快捷键，并且自动判断是英语还是汉语，并进行英译汉或者英译汉
##### 翻译采用百度的API来进行 

### 编译
##### windows下编译exe文件可以使用pyinstaller来进行
##### **pyinstaller.exe -w -F clipboard.py**
